== Description ==

Shoppable’s patented, universal checkout plugin allows you to host a single multi-retailer shopping bag and marketplace on your website. Our technology transforms your content into Shoppable® content. This full-service e-commerce technology is used by everyone from bloggers to digital media companies to internationally known consumer companies. Now you can monetize your site through an e-commerce affiliate checkout without ever having to forward your reader to a retailers website. [Click here to sign up for a Shoppable account](https://www.shoppable.com/new_sign_up), and then install this plugin to get started.

== Installation ==

1. Install and activate the Shoppable plugin from your Wordress Dashboard Plugins page.
1. Sign up with Shoppable and get your token [here](https://www.shoppable.com/new_sign_up).
1. Enter your token and additional settings in the Wordress Dashboard Settings/Shoppable page.

= After Activation =

1. You should see the marketplace in the element that you selected in the settings and the mini-cart tab should be visible in the top right corner of your page
1. If desired, add additional CSS to the wp-content/plugins/shoppable/assets/css/marketplace.css file
1. You're done!
